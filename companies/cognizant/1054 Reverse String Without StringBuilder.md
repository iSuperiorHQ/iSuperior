# **Problem 1054: Reverse a String Using Raw Loops**

**Company:** Cognizant

**Topic:** Strings

---

## **Problem Description**

You are given a string. Your task is to **reverse the string manually** using basic programming constructs.

You are **not allowed to use built-in utilities** such as `StringBuilder`, `reverse()` methods, or any library functions for reversing.

---

## **Task**

* Reverse the given string using **loops only**
* Construct the reversed string character by character

---

## **Input Format**

* A single line containing the string

---

## **Constraints**

* **1 ≤ length of string ≤ 10⁵**
* The string may contain alphabets, digits, and special characters

---

## **Output Format**

* Print the **reversed string**

---

## **Sample Input 1**

```text id="z6k2mv"
hello
```

---

## **Sample Output 1**

```text id="u4p9zx"
olleh
```

---

## **Explanation**

Original string:

```text id="d7m1qp"
hello
```

Reversed string:

```text id="n3x8vk"
olleh
```

---

## **Sample Input 2**

```text id="t5k9ra"
abc123
```

---

## **Sample Output 2**

```text id="m8z2xp"
321cba
```

---

## **Explanation**

Each character is reversed using loop traversal.

---

## **What they check:**

* Understanding of **loops and indexing**
* Ability to manipulate strings manually
* Avoiding built-in shortcuts

---

## **Execution Time Limit**

**10 seconds**
